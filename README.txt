How to compile and view run times:

1. Open program1.cpp in desired IDE.

2. From the terminal do the following: g++ -Wall program1.cpp -o make

3. Then, type in: ./make

4. You will then be prompted by the program to enter an integer. (10, 100, 1000, 10000)

5. You are prompted a second question by the program to enter any positive integer.

6. You will be notified to go to your directory to view the output text file.

7. Check all run times in the output text file.
